განსხვავება ცარიელსა და კონტეინერ თეგებს შორის

ცარიელი თეგი - აქვს მხოლოდ გამხსნელი ტეგი.


კონტეინერი თეგი - აქვს როგორც გამხსნელი, ისე დახურვის თეგი.

Inline და Block level ელემენტებს შორის განსხვავება


Inline Element - ელემენტი საიტზე იკავებს იმდენ ადგილს რამდენიც სჭირდება.


Block level Element - ელემენტი საიტს იკავებს მთლიან სიგანეზე.

რა არის ატრიბუტი

ატრიბუტი - ატრიბუტი არის ის, რაც თეგის შიგნით წერია და ამ თეგს ფუნქციას, მნიშვნელობას აძლევს.
