ჩვენი კურსი

GOA-ში შეისწავლით სხვადასხვა პროგრამირების ენებს:

  • HTML - საიტის სტრუქტურის შექმნა
  • CSS - საიტის დიზაინის შექმნა
  • Python - მარტივი და ძლიერი ენა, რომლითაც შეგიძლია შექმნა პროგრამები, საიტები, თამაშები, ხელოვნური ინტელექტი და სხვა
  • JavaScript - გამოიყენება ვებ–გვერდებზე ფუნქციების შესაქმნელად
  • React - JavaScript-ის ბიბლიოთეკაა, და გამოიყენება ლამაზი, სწრაფი და თანამედროვე საიტების შესაქმნელად
